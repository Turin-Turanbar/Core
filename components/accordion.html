Components

Accordion

A collapsible container that reveals or hides content when toggled. Use it to progressively disclose information and reduce visual complexity.

Collapsed

The default state. The header is visible; content is hidden. The + icon signals there is more to reveal.

Slot
<core-accordion
  [headline]="'Template Information'"
  [isOpen]="false"
  [hasBadge]="false">
</core-accordion>

Expanded

The open state. Content is revealed in the slot below the header. The icon transitions to − to indicate the item can be collapsed.

Slot
<core-accordion
  [headline]="'Template Information'"
  [isOpen]="true"
  [hasBadge]="false">
  <!-- projected content goes here -->
</core-accordion>

With Badge

Set [hasBadge]="true" to display a numeric badge next to the headline. Useful for surfacing counts — pending items, required fields, etc.

Slot
<core-accordion
  [headline]="'Template Information'"
  [isOpen]="false"
  [hasBadge]="true">
</core-accordion>

With Badge Expanded

Badge and open state combined. The badge remains visible in the header while the content slot is displayed below.

Slot
<core-accordion
  [headline]="'Template Information'"
  [isOpen]="true"
  [hasBadge]="true">
  <!-- projected content goes here -->
</core-accordion>

Props

Prop Type Default Description
headline string "Template Information" Label shown in the accordion header.
isOpen boolean false Controls whether the content slot is visible.
hasBadge boolean false Shows a numeric badge (value: 1) next to the headline.
