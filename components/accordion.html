Accordion

A collapsible container that reveals or hides content when toggled. Use it to progressively disclose information and reduce visual complexity.

Default

Click the header to toggle between collapsed and expanded. Use [isOpen] to set the initial state.

Place any content here — descriptions, forms, settings, or nested components.

<core-accordion
  [headline]="'Template Information'"
  [isOpen]="false"
  [hasBadge]="false">
  <your-content />
</core-accordion>

With Badge

Set [hasBadge]="true" to show a numeric badge beside the headline — useful for counts like pending items or required fields.

Place any content here — descriptions, forms, settings, or nested components.

<core-accordion
  [headline]="'Template Information'"
  [isOpen]="false"
  [hasBadge]="true">
  <your-content />
</core-accordion>

Props

PropTypeDefaultDescription
headline string "Template Information" Label shown in the accordion header.
isOpen boolean false Controls whether the content slot is visible.
hasBadge boolean false Shows a numeric badge next to the headline.
