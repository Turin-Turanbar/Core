Foundation

Tokens

Design tokens are the single source of truth for all visual decisions in CORE — colors, typography, spacing, and shape. They're defined in Figma and exposed as CSS custom properties (--core-*).

Color

The color system is organized into two layers: primitives (the raw palette) and semantic tokens (purpose-driven aliases that reference primitives).

The primary neutral scale. neutral/950 is designated as the primary brand color.

Typography
